This is a networking framework I wrote to learn about Unreal Engine 5, and how its networking tools compare to Unity's Networking for GameObjects. Here's a list of some of its features.

* Networked synchronized FSM, driving animations and core client and server logic.
* Voice communication via EOS (Epic Online Services).
* FinState implementation with separate virtual methods for client and server actions.
* Simple Lobby browser UI that can handle multiple games on one LAN network.
